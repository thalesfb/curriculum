## Criando uma nova ADR

1. Copie o template de uma ADR existente
2. Nomeie como `NNNN-titulo-com-tracos.md`
3. Preencha contexto, decisão e consequências
4. Atualize este índice

## Status possíveis

- **Proposed**: Em discussão
- **Accepted**: Decisão tomada, implementando
- **Deprecated**: Não mais relevante
- **Superseded**: Substituída por outra ADR
